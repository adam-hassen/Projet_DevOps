# Application de Gestion de Clients et Commandes

## Description
Application web pour gérer des clients et leurs commandes.

## Fonctionnalités
- Gestion des clients
- Gestion des commandes
- Génération de factures

## Comment lancer ?
1. Installer Python 3.9+
2. Installer les dépendances : `pip install -r requirements.txt`
3. Lancer : `python app.py`
4. Ouvrir http://localhost:5000

## Structure
- `app.py` : Application principale
- `templates/` : Pages HTML
- `static/` : Fichiers CSS/JS
- `commandes.xlsx` : Base de données
